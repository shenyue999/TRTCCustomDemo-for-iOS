### 说明 

* **本项目只用于学习和交流，实时腾讯音视频TRTC采集与渲染接口使用Demo

* **下载项目 若需要真实进房 需要在GenerateTestUserSig.h填写你的_SDKAppID与_SECRETKEY

* 可以下载官方Demo填写与工程相同的_SDKAppID与_SECRETKEY 进房查看远端接收效果


### 编译环境

Xcode + MacOS 本工程只保留TRTC SDK arm64架构,可以下载官方的SDK进行替换
